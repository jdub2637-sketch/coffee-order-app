# Development Plan - Coffee Order App

## Phase 1: Environment Setup
- [ ] Vite를 사용한 React 프로젝트 스캐폴딩.
- [ ] 기본 폴더 구조 정리 (components, assets, styles).
- [ ] 필요한 아이콘 또는 배경 이미지 준비 (Placeholder 활용).

## Phase 2: Design Implementation (Glassmorphism)
- [ ] 전역 스타일링 (배경 이미지, Blur 효과).
- [ ] 주문 카드 및 입력 폼 컴포넌트 디자인.
- [ ] 반응형 레이아웃 확인 (모바일 뷰).

## Phase 3: Core Logic Development
- [ ] Admin 기능: 메뉴 및 가격 관리 상태 로직 구현.
- [ ] User 기능: 이름 입력 및 메뉴 선택 상태 관리.
- [ ] API 연동: Fetch API를 사용한 Webhook 전송 함수 작성.

## Phase 4: Integration & Test
- [ ] Admin에서 등록한 메뉴가 User 화면에 실시간 반영되는지 확인.
- [ ] 주문 시 가상 URL로 데이터가 정상적으로 전달되는지 테스트 (Console Log).
- [ ] 전체적인 UI/UX 흐름 검토.

## Phase 5: Deployment Ready
- [ ] Webhook URL 변수화 확인.
- [ ] 빌드 및 실행 방법 안내.
